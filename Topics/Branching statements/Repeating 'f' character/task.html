Repeating 'f' character

How many times will the letter 'f' be printed?

for (int i = 0; i < 5; i++) { 
    for (char letter = 'a'; letter <= 'f'; letter++) { 
        if (letter < 'f') { 
            continue; 
        } 
        System.out.println(letter); 
    } 
}

The answer should be a single number.
